Credentials live in the server's .env file and are never editable from the admin UI. To change the bucket or rotate keys, edit .env and restart the server.

Setup Guide — Email Delivery
1
Choose Email Provider
Popular SMTP options: Gmail (smtp.gmail.com:587), SendGrid (smtp.sendgrid.net:587), Mailgun (smtp.mailgun.org:587), AWS SES.
2
Gmail App Password Setup
Enable 2FA first. Then myaccount.google.com → Security → App Passwords → generate 16-char password.
3
Test Your Configuration
After saving, use Send Test Email button. Check host, port, and credentials if it fails.
